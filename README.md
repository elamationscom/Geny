Geny – Social AI Scheduler (Demo)

Quickstart (Next.js)

- Prerequisites: Node.js 18+ and npm
- Install: `npm install`
- Dev: `npm run dev` (http://localhost:3000)
- Build: `npm run build` then `npm start`

What’s included

- Next.js App Router + TypeScript + Tailwind
- Clean, minimal UI with glass/mesh aesthetics
- Pages: Landing, Auth (mock), Dashboard, Compose (with AI suggestions), Calendar, Settings
- Local persistence via Zustand (no backend yet)
- Mock AI suggestion generator (replace with your provider later)

Next steps
